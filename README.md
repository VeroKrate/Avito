# Автоматизированное тестирование API микросервиса объявлений

## Описание проекта

Проект содержит автоматизированные тесты для проверки API микросервиса объявлений Avito. Тесты охватывают основные операции:
- Создание объявлений
- Получение объявлений по ID
- Получение статистики по объявлениям

**Базовый URL API:** `https://qa-internship.avito.com`

## Предварительные требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- Доступ к интернету (для запросов к API)
