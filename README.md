# Тестирование API микросервиса объявлений
Инструкция по запуску автоматизированных тестов.

## Предварительные требования
*   Установленный Python версии 3.8 или выше.
*   Менеджер пакетов pip.

## Установка и настройка
1.  Склонируйте репозиторий или скопируйте файлы проекта.
2.  Установите необходимые зависимости, выполнив в терминале команду: pip install -r requirements.txt

## Запуск тестов
Для запуска всех автоматизированных тестов выполните в корневой директории проекта команду: pytest test_api.py -v

Структура проекта
TESTCASES.md - тест-кейсы
README.md - инструкция
BUGS.md - баг-репорты
test_api.py - автоматизированные тесты
requirements.txt - зависимости
